## 基本信息

| 项目 | 内容 |
| --- | --- |
| 文件 | `scripts/quark.py` |
| cron | `50 8 * * *` |
| 任务名 | `夸克网盘` |
| 依赖 | Python `requests`、青龙 `notify.py` |
| 环境变量 | `QUARK_COOKIE` |

## 环境变量

`QUARK_COOKIE` 填夸克网盘移动端 Cookie，需要包含 `kps`、`sign`、`vcode`。多账号优先换行分隔。

## 执行流程

1. 检查 Cookie 是否包含移动端参数。
2. 获取成长信息和签到状态。
3. 未签到时执行签到。
4. 通知中展示奖励空间、连签进度、总空间和累计签到奖励。

## 常见失败

| 现象 | 可能原因 | 处理 |
| --- | --- | --- |
| Cookie 缺少移动端参数 | 抓到的是网页端 Cookie 或复制不完整 | 从移动端相关请求重新抓取 |
| 获取成长信息失败 | Cookie 失效或接口返回异常 | 重新登录并更新 Cookie |
| 签到失败 | 已触发风控或接口结构变化 | 查看接口返回 message 后再判断是否要更新脚本 |
