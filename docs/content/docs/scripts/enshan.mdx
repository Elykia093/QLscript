## 基本信息

| 项目 | 内容 |
| --- | --- |
| 文件 | `scripts/enshan.py` |
| cron | `20 8 * * *` |
| 任务名 | `恩山无线论坛` |
| 依赖 | Python `requests`、青龙 `notify.py` |
| 环境变量 | `ENSHAN_COOKIE` |

## 环境变量

`ENSHAN_COOKIE` 填恩山论坛登录后的完整 Cookie。多账号优先换行分隔。

## 执行流程

脚本访问积分页面，解析恩山币和积分，再通过通知汇总每个账号结果。

## 常见失败

| 现象 | 可能原因 | 处理 |
| --- | --- | --- |
| 无法提取恩山币或积分 | Cookie 失效，或页面结构变化 | 重新抓 Cookie；若仍失败，需要更新解析规则 |
| HTTP 403/跳登录 | Cookie 不完整或过期 | 重新登录后复制完整 Cookie |
| 账号显示为账号序号 | Cookie 中没有可识别字段 | 不影响运行，只影响通知标题 |
