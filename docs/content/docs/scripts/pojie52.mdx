## 基本信息

| 项目 | 内容 |
| --- | --- |
| 文件 | `scripts/pojie52.py` |
| cron | `5 9 * * *` |
| 任务名 | `吾爱破解` |
| 依赖 | Python `requests`、青龙 `notify.py` |
| 环境变量 | `POJIE52_COOKIE` |

## 环境变量

`POJIE52_COOKIE` 填吾爱破解论坛登录后的完整 Cookie。多账号优先换行分隔。

## 执行流程

1. 访问签到页面。
2. 如果已经签到，直接查询积分。
3. 未签到时解析 `formhash` 并提交签到。
4. 签到完成后查询积分，通知中展示吾爱币、积分、热心值或技术成就等信息。

## 常见失败

| 现象 | 可能原因 | 处理 |
| --- | --- | --- |
| 未找到 formhash | Cookie 失效或页面结构变化 | 重新登录后抓取完整 Cookie |
| 签到失败且返回 HTML 文案 | 账号状态、风控或论坛规则变化 | 查看清理后的错误文本 |
| 未获取到积分信息 | 积分页结构变化 | 更新积分解析规则 |
