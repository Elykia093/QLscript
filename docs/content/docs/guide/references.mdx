## 本仓库规范

新增脚本以 `SCRIPT_STANDARD.md` 为准。文档站的脚本索引、根 README、`scripts/README.md` 和脚本头注释需要同步维护。

## Fumadocs 文档站

本文档站使用 Next.js App Router、Fumadocs UI、Fumadocs MDX 和静态导出。源码在 `docs/`，内容在 `docs/content/docs/`。
