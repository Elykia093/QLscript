## 通知行为

Python 脚本优先调用青龙 `notify.py`，JavaScript 脚本优先调用青龙 `sendNotify.js`。通知模块不存在时会降级为控制台输出。

## 失败隔离

脚本应按账号逐个执行。单个账号失败不应阻断其他账号，最终日志和通知需要汇总成功数、失败数和关键原因。

## 详细说明

每个脚本都有独立说明页，包含凭据要求、执行流程和常见失败原因。新增脚本时同步新增脚本详解页，并在本页链接到它。
